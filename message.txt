VAR, LET e CONST

2º Utilize let para declarar uma variável que armazena a soma de dois números. Altere o valor de um dos números e exiba a nova soma.
3º Declare uma constante (const) e atribua a ela um array de números. Tente adicionar um novo número ao array e observe o que acontece.
4º Utilize var para declarar uma variável dentro de um bloco ({}) e tente acessá-la fora desse bloco. Faça o mesmo com let.
5º Declare uma variável usando let dentro de um loop for e tente acessá-la fora do loop.
6º Crie uma função que utiliza const para declarar uma constante internamente e tenta modificá-la. O que acontece?
7º Declare uma variável utilizando var fora de uma função e tente acessá-la dentro de uma função. Repita o processo com let.
8º Utilize const para declarar um objeto. Adicione uma propriedade a esse objeto.
9º Declare uma variável com var e outra com let dentro de um bloco if. Tente acessá-las fora do bloco.
10º Crie uma função que recebe um parâmetro e utiliza const para declarar uma variável local que armazena o dobro desse parâmetro.
